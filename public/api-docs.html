üìù Quick Start Guide

  • Server Laravel sudah berjalan di http://127.0.0.1:8000
  • Gunakan tools seperti Postman, Thunder Client, atau cURL untuk testing
  • Mulai dengan Register ‚Üí Login ‚Üí Dapatkan Token ‚Üí Test endpoints lainnya
  • Token JWT harus disertakan di header Authorization: Bearer {token} untuk endpoint yang dilindungi

üîì Public Endpoints

POST /api/register PUBLIC

Mendaftarkan user baru

curl -X POST http://127.0.0.1:8000/api/register \ -H "Content-Type: application/json" \ -d '{ "name": "John Doe", "email": "john@example.com", "password": "password123", "password_confirmation": "password123" }'
POST /api/login PUBLIC

Login dan mendapatkan JWT token

curl -X POST http://127.0.0.1:8000/api/login \ -H "Content-Type: application/json" \ -d '{ "email": "john@example.com", "password": "password123" }'
Response akan berisi:
{ "access_token": "eyJ0eXAiOiJKV1QiLCJhbGc...", "token_type": "bearer", "expires_in": 3600 }

üìù Post Endpoints

GET /api/posts/ AUTH REQUIRED

Mendapatkan semua posts

curl -X GET http://127.0.0.1:8000/api/posts/ \ -H "Authorization: Bearer YOUR_TOKEN_HERE"
GET /api/posts/my_posts AUTH REQUIRED

Mendapatkan posts milik user yang sedang login

curl -X GET http://127.0.0.1:8000/api/posts/my_posts \ -H "Authorization: Bearer YOUR_TOKEN_HERE"
POST /api/posts/create AUTH REQUIRED

Membuat post baru

curl -X POST http://127.0.0.1:8000/api/posts/create \ -H "Content-Type: application/json" \ -H "Authorization: Bearer YOUR_TOKEN_HERE" \ -d '{ "title": "My First Blog Post", "body": "This is the content of my blog post..." }'
POST /api/posts/posts/{id} AUTH REQUIRED

Update post berdasarkan ID

curl -X POST http://127.0.0.1:8000/api/posts/posts/1 \ -H "Content-Type: application/json" \ -H "Authorization: Bearer YOUR_TOKEN_HERE" \ -d '{ "title": "Updated Title", "body": "Updated content..." }'
DELETE /api/posts/{id} AUTH REQUIRED

Hapus post berdasarkan ID

curl -X DELETE http://127.0.0.1:8000/api/posts/1 \ -H "Authorization: Bearer YOUR_TOKEN_HERE"
POST /api/posts/like AUTH REQUIRED

Like/Unlike post

curl -X POST http://127.0.0.1:8000/api/posts/like \ -H "Content-Type: application/json" \ -H "Authorization: Bearer YOUR_TOKEN_HERE" \ -d '{ "post_id": 1 }'
